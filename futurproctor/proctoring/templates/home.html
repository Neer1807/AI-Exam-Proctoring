Advanced Proctoring Features

Comprehensive Monitoring for Exam Integrity

Face verification

The System opens the exam only after facial verification. The student uploads his picture during registration, and it is used in all his exams to make sure he doesn't leave the examination place.

Browser Tab Switching Detection

This system will monitor tab switching and flag any unauthorized attempts during the exam.

Object Detection

This system will detect unauthorized objects like books and smartphones during the exam.

Gaze Detection

This system will detect gaze movement to ensure candidates stay focused on the exam screen.

Multiple Person Detection

This system will detect the presence of multiple individuals during the exam to ensure proper authentication.

Audio Analysis

This system will record and analyze background sounds to detect any suspicious noises during the exam.

Secure Data Management: All collected data is securely stored in PostgreSQL for comprehensive review and reporting.
